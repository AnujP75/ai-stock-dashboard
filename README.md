# ai-stock-dashboard
Real-time AI-powered stock market insight dashboard

## Data Sources & Rights

This dashboard uses publicly available data from:

- Yahoo Finance (via yfinance library) for stock market data
- NewsAPI for real-time financial news headlines

All data is used in accordance with their respective terms of service:
- Yahoo Finance data is for personal and educational use
- NewsAPI data is used under the free developer tier

This project is intended for educational and non-commercial use only.

## Disclaimer

This tool does not provide financial advice. All insights are generated using basic models and publicly available data. Users should conduct their own research before making investment decisions.
